# mithril-stopwatch

A lightweight World of Warcraft interface addon designed to expand existing stopwatch functionality and include support for timers.

## Features

* **Create Timers and Stopwatches:** Add as many of either as you need, all will run independent of each other.
* **Rename Anything:** Supports ability to rename any active frames for specific activities or events.
* **Rebuilt Stopwatch Functionality:** Running a stopwatch is no longer dependent on played time. Stopwatches now continue account-wide even if you are logged out!

## Installation

Download the latest release from the Releases page or use WowUp's `Install from URL` functionality for automatic updating.

## Usage

* Toggle visibility with the icon attached to the `Objective Tracker` frame
* Add additional timers and stopwatches with the `+` button
* Double-click the headers to rename
